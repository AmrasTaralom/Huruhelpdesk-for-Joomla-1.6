Huru Helpdesk User Sync Plugin 
v0.8x.01

NOTE: This version of the plugin should only be used with Huru Helpdesk version 0.8x (0.86, 0.87, etc.)

*** Installation & Configuration ***
1) Install the plugin via the Joomla installer.
2) From the Extensions menu, select Plugin Manager
3) Find and click on "Huru Helpdesk - User Sync"
4) Set the "Assign 'User' level by default?" parameter if you want imported Joomla users to automatically
   be given User access to the helpdesk
5) Enable the plugin
6) Click Save
7) Test


*** Known issues ***
- English-only for now (if you want to customize the xml file for your own language - go ahead)
